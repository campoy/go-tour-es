Cómo puedes ayudar

Reportando errores

Si descubres bugs, errores, o inconsistencias en el código o la documentación del proyecto, por favor, háznoslo saber rellenando un ticket en nuestro gestor de errores. (Por supuesto, deberías comprobar que no existe otro error igual antes de crear uno nuevo).

Nos enorgullecemos de ser meticulosos; ningún error es pequeño.

Contribuyendo con código

Go es un proyecto open source y damos la bienvenida a las contribuciones de la comunidad.

Para empezar, lee esta guía de contribución para informarte sobre el diseño, testing y el proceso de revisión de código.

Revisa el gestor para ver errores abiertos que te puedan interesar. Aquellos etiquetados como HelpWanted tienen un interés particular en necesitar ayuda externa.
